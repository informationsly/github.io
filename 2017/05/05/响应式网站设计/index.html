响应式网站设计

响应式网站设计的概念

响应式网站是一个设计概念，它是多项技术的综合体

由来

2010年Ethan Marcotte在Responsive Web Design一文中提出了响应式网站设计的概念

用到的技术

  • flexible grid layout 弹性(流式)网格布局
  • flexible image 弹性图片
  • media queries 媒体查询（css2.1出现的概念，css3被强化）

优点

  • 每个设备都能得到正确的设计
  • 搜索优化
  • 更好的用户体验

缺点

  • 会加载更多的样式和脚本资源
  • 设计比较难精确定位和控制
  • 老版本浏览器兼容不好

视口

布局视口

布局视口为网页PC端版式，为默认布局模式下的页面显示大小

可视视口

可视视口为手机端能看见PC端网页的显示大小（默认是设备的宽度）：如同放大镜离书本距离变化后，放大镜里能看到的内容的多少，不能简单理解为设备屏幕的大小。

理想视口

理想视口就是为构建手机浏览器优化的页面而添加的。
理想视口（就是布局视口在一个设备上的最佳尺寸）应该理解为一个状态值，

网页中的单位

  • px
    1个px相当于1个像素

  • em
    相对的长度单位(em依赖与父元素的字体大小)
    1.em 相对参照物为父元素的font-size
    2.em 具有继承的特点
    3.当继承链上所有的父元素都没有设置font-size时，浏览器会有一个默认的em设置：1em = 16px
    em的缺点：各个em元素的父元素的字体大小不确定，容易造成混乱

  • rem
    1.rem的相对参照物为根元素html,相对参照固定不变，所以比较好计算
    2.当没有html没有设置font-size时，浏览器会有一个默认的rem设置：1rem = 16px，这点与em是一致的
    3.IE8及以下的IE并不支持rem属性

当font-size设置为62.5%(10/16*100%)时，1rem = 10px, 便于计算
font-size:62.5% 1rem = 10px
font-size:100% 1rem = 16px
